NP Express Generator
====================

##Includes:##

* Validations
* Config files
* Mongoose
* Language files

##How to install:##
```
npm install -g yo;
npm install -g generator-np-express;

cd /Volumes/Development && mkdir your-app;
cd !$;

yo np-express;
```

##How to use:##
```
cd your-app;
npm start;
```
